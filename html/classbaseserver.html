Detailed Description

Main server class./

Main server class. Responsible for implementation of functions needed by Libevent to properly handle connections. Implements only basic functions of a server, such as sending and writing data and registering new clients but do not archive the data about them. This is the responsibility of a class [serverlogic]: serverlogic
